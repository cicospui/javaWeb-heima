设置颜色时的三种表示形式，推荐内嵌方式写在style中：

```html
<style>
    h1{
        三种颜色定义方式
        color: black;			单词
        color: rgb(1, 1, 1);	rgb表示
        color: #00f;			十六进制
    }
</style>
```

并使用**span标签**对一行的内容进行分别处理。
